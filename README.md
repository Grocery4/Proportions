# Proportions
Small program to test GitHub that returns measures in another scale.

This program should be able to convert:

Time:
    Seconds
    Minutes
    Hours
    Days
    Weeks
    Months
    Years

Length:
    Metre(s) and all of its multiple dividers


Mass:
    Gram(s) and all of its multiples/dividers

Temperature:
    Kelvin
    Centigrades
    + Farenheit
